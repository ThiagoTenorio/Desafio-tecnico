# Desafio Técnico

Este repositório contém soluções para um desafio técnico com cinco questões. As primeiras três questões foram resolvidas com código JavaScript, e as duas últimas foram respondidas em forma de comentários.

## Como Acessar o Projeto

Para acessar as soluções do projeto, você precisa clonar o repositório em sua máquina local. Utilize o seguinte comando no terminal:

```bash
git clone https://github.com/ThiagoTenorio/Desafio-tecnico
```

## Como Executar os Códigos

Após clonar o repositório, você pode executar os códigos JavaScript utilizando o Node.js. Para fazer isso, siga os passos abaixo:

1. Navegue até o diretório onde o repositório foi clonado:
   ```bash
   cd Desafio-tecnico
   ```

2. Execute o arquivo JavaScript desejado usando o seguinte comando:
   ```bash
   node <nome-do-arquivo>.js
   ```

   Substitua `<nome-do-arquivo>` pelo nome do arquivo que você deseja executar.

## Questões e Respostas

### Questão 1: Sequência de Fibonacci
- **Descrição:** O programa calcula a sequência de Fibonacci e verifica se um número fornecido pelo usuário pertence à sequência.
- **Arquivo:** `questao1.js`

### Questão 2: Contagem de Letras
- **Descrição:** O programa verifica a existência da letra ‘a’ em uma string, contando quantas vezes ela aparece.
- **Arquivo:** `questao2.js`

### Questão 3: Soma de Números
- **Descrição:** O programa realiza uma soma de números de 1 até um índice fornecido e exibe o resultado.
- **Arquivo:** `questao3.js`

### Questão 4: Identificação de Lógica
- **Resposta:** Apresentei um método para resolver um problema lógico envolvendo números.

### Questão 5: Interruptores e Lâmpadas
- **Resposta:** Descrevi a lógica de como descobrir qual interruptor controla cada lâmpada em duas idas até a sala.
